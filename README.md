# Salary Prediction Pipeline (SE Factory - Week 1)

An end-to-end ML pipeline that predicts Data Science salaries and generates insights using a local LLM.

## Tech Stack
- **Environment Management:** `uv`
- **Machine Learning:** Scikit-Learn (Decision Tree)
- **API:** FastAPI
- **Database:** Supabase
- **Dashboard:** Streamlit
- **LLM:** Ollama (Llama 3.1)

## Setup
1. Install `uv`: `curl -LsSf https://astral.sh/uv/install.sh | sh`
2. Sync environment: `uv sync`
3. Run Local LLM: `ollama run llama3.1`
